## 腾讯云函数收费了，项目已修改为 github actions 执行

在项目 actions->secret 中添加如下几个变量：

- EMAIL_USER 发送邮件的邮箱账号
- EMAIL_PASS 发送邮件的 token
- USER_MOBILE 掘金账号 - 手机号
- USER_PASSWORD 掘金账号 - 密码
- USER_EMAIL 接收通知的邮箱账号
